# Production Manual Test Checklist

Vercel + Neon の第1段階本番テスト用チェックリストです。  
対象URLは `https://hunting-app-test.vercel.app` です。

## 前提

- 画像アップロードは今回の確認対象外。
- 本番DB構造変更、`prisma:push`、Vercel / Neon 設定変更は行わない。
- 固定デモユーザー前提で、主要ページ表示と画像アップロード以外のCRUDを確認する。
- 確認後、結果を `TODO.md` / `CHANGELOG.md` に別途記録する。

## 確認順序

1. トップページ表示
2. 更新管理 `/renewals`
3. 実包帳簿 `/ammo`
4. 活動記録 `/reports`
5. 利用者設定 `/account`
6. ダッシュボード再確認
